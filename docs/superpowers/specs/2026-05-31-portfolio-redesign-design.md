## 1. 개요

기존의 다크 + 네온(#bcff00) + Three.js "기괴한 미학" 콘셉트 사이트를, **젠틀몬스터(gentlemonster.com) 스타일의 화이트톤 에디토리얼**로 전면 교체한다. 상단에 풀스크린 영상 히어로를 두고, 그 아래로 실제 프로젝트 7개·프로필·연락처를 단일 페이지 스크롤로 담는다.

핵심 전환:
- **다크 → 화이트톤**(웜 화이트 배경 + 잉크 블랙 텍스트, 모노크롬, 액센트 색 없음)
- **콘셉트 작품 → 실제 포트폴리오**(허유경의 진짜 프로젝트 7개)
- **Three.js 크롬 씬 제거** → 영상 + 사진 + 스크롤 모션 기반 인터랙션
- 단일 HTML 정적 사이트 유지, **GitHub Pages 배포**

기존 CLAUDE.md의 "다크 네온 디자인 시스템 유지(do not drift)" 항목은 본 리디자인으로 **대체**된다. 구현 완료 후 CLAUDE.md를 새 디자인 시스템으로 갱신해야 한다.

## 6. 기술 / 파일 / 배포

- **단일 `index.html`** + CDN(Tailwind, GSAP+ScrollTrigger, Lenis). 빌드 시스템·번들러·npm 없음(기존 제약 유지).
- 영상/이미지는 repo에 동봉(영상 2.9MB, GitHub 100MB/파일 제한 한참 아래라 LFS 불필요).
- 기존 `portfolio.html`의 다크/Three.js/네온 자산은 제거하고 새로 구축. `portfolio.html` → `index.html`로 전환(Pages 기본 파일명).
- **배포: GitHub Pages**(무료·자동 HTTPS). 추후 S3+CloudFront로 이전해도 동일 파일 그대로 사용 가능.
